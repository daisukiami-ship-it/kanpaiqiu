## 前置：注册 + 买域名
1. 注册 Cloudflare：https://dash.cloudflare.com/sign-up
2. 买域名：Dashboard → Domain Registration → Register Domains（$10/年 .com，需信用卡/PayPal）
   - 或国内买（腾讯云/阿里云，支持支付宝），买完把 DNS 托管到 Cloudflare（改 nameserver）。

---

## 部署方式 A：Dashboard 直接拖拽（最简单，推荐）
1. Dashboard → 左侧 **Workers & Pages** → **Create** → **Pages** → **Upload assets**
2. 项目起名（如 `vbplayer`）
3. 把 `cloudflare/` 目录里的**内容**（index.html + assets/ + functions/）打包上传
   - 注意：上传的是目录**内容**，不是外层 cloudflare 文件夹本身
4. 部署完成，得到临时地址：`https://vbplayer.pages.dev`

## 部署方式 B：命令行 Wrangler（适合以后自动化）
```
npm i -g wrangler
cd cloudflare
wrangler pages deploy . --project-name=vbplayer
# 环境变量：
wrangler pages secret put YOUTUBE_API_KEY --project-name=vbplayer
```

## 绑定自定义域名
项目 → **Custom domains** → **Set up a domain** → 输入你买的域名（如 `yourdomain.com`）
- 域名已托管在 Cloudflare → 自动加 DNS 记录，几分钟生效
- HTTPS 证书 Cloudflare 自动签，无需手动配

访问：`https://yourdomain.com/`（如果 index.html 在根目录）

## 与 Pantheon 版的差异（重要）
- **YouTube Key**：Pantheon 放 files/private/secrets.json；CF 放环境变量 YOUTUBE_API_KEY。
- **频道白名单**：Pantheon 从 secrets.json 读；CF 内置在 live-vb.js 常量（也可用环境变量覆盖）。
- **TRT**：两版都是超链接跳官网（CORS+地域双限，无法内嵌），未改动。
- **proxy**：CF 版备用；注意 CF 边缘 IP 全球分布，对 TRT 的 CloudFront 地域锁同样可能 403。

## API Key 安全提醒
- Key 只存在于服务端环境变量 / secrets.json，前端 index.html 不含 Key（已验证无 bing 外链、无硬编码 Key）。
- 建议在 Google Cloud Console 给该 Key 加 **HTTP referrer 限制** 或 **API 限制**（只允许 YouTube Data API v3），降低泄露风险。
