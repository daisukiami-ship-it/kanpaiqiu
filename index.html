使用说明

  1. 复制你的 M3U8 流地址（通常以 .m3u8 结尾）或 YouTube 链接。
  2. 将地址粘贴到上方输入框。
  3. 点击 播放 按钮，HLS 播放器会在浏览器中直接加载并播放。
  4. 若流需要特定来源校验，可在高级选项中填写 HTTP Referrer 或自定义 User Agent。

支持：M3U8/HLS 流、MP4 直链、YouTube 视频。右侧为 YouTube 上正在进行的排球直播，点击即可在此处播放。浏览器需开启 CORS 且服务端允许跨域。 点击获取视频源
